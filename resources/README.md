# Kangentic brand resources (v2, Warm Craft)

The CURRENT selected mark, cut per surface. Generated by
`npm run gen:icons` - regenerate, never hand-edit. Two-tier rule: sizes
below 48px render the F4k board glyph, 48px and up the card-K; ICO/ICNS
mix tiers by entry. Knockout renditions carry true alpha holes (the theme
or wallpaper shows through, like the v1 blue K did).

## desktop/ (kangentic, Electron)
| File | Use |
|------|-----|
| icon.ico | Windows app icon (taskbar, title bar, installer); 16-256, mixed tiers |
| icon.icns | macOS dock; 16-1024, mixed tiers |
| icon.png + icon-{16..1024}.png | Linux / BrowserWindow ladder (knockout) |

## web/ (kangentic.com)
| File | Use |
|------|-----|
| favicon.ico, favicon-16x16.png, favicon-32x32.png | browser tabs (F4k knockout) |
| apple-touch-icon.png (180) | iOS home-screen bookmark (opaque cream square) |
| icon-192.png, icon-512.png | site.webmanifest (opaque cream square) |
| logo.png (512) | JSON-LD publisher logo (card-K knockout) |
| brandmark.svg / brandmark-small.svg | in-page marks (>=48px / <48px) |
